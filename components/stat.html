Stat & queue card

Stat is a single large counter with a label. Use stats for dashboard summaries. Queue card wraps a stack of stats with a header and an "Open queue" CTA at the bottom.

Stat

Variants: stat-alert (red), stat-success (green), stat-warning (orange).

7 Unassigned articles
14 In peer review
3 Decisions ready to send 
<div class="stat stat-alert">
  <span class="stat-number">7</span>
  <span class="stat-label">Unassigned articles</span>
</div>
<div class="stat">
  <span class="stat-number">14</span>
  <span class="stat-label">In peer review</span>
</div>
{% raw %}{{ stat(7,  "Unassigned articles", kind="alert") }}
{{ stat(14, "In peer review") }}
{{ stat(3,  "Decisions ready to send", kind="success") }}{% endraw %}

Queue card

Bordered card with header, stat stack, and a CTA at the foot. Used to represent a single workflow queue on the dashboard.

Editor queue

Open queue 
<section class="queue-card" aria-labelledby="qc-1">
  <div class="queue-card-head">
    <h2 class="queue-card-title" id="qc-1">Editor queue</h2>
    <span class="tag tag-grey">Editor</span>
  </div>
  <div class="queue-card-body">
    <div class="stat stat-alert"> … </div>
    <div class="stat"> … </div>
    <div class="stat stat-success"> … </div>
  </div>
  <div class="queue-card-foot">
    <a href="#" class="btn">Open queue</a>
  </div>
</section>
{% raw %}{% call queue_card("Editor queue", tag={"label": "Editor", "kind": "grey"}, cta={"label": "Open queue", "url": "/editor"}) %}
  {{ stat(7,  "Unassigned articles", kind="alert") }}
  {{ stat(14, "In peer review") }}
  {{ stat(3,  "Decisions ready", kind="success") }}
{% endcall %}{% endraw %}

Notes

  • Stats stack vertically inside a queue card. Each consecutive stat gets a top border to separate the rows.
  • Numbers must be integers, not abbreviated (use 14, not 14k) — editorial dashboards deal in small absolute counts.
