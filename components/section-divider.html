Section divider

A thin heading line that separates groups within a flat list. Used for severity buckets on the inbox dashboard, category groupings on the manager, and date-stamp headings in timelines.

Default

Heading text on the left, a thin line filling the rest of the row.

Journal settings · 5

Review · 2

<div class="section-divider section-divider-flush">
  <h2 class="section-divider-label">Journal settings · 5</h2>
  <span class="section-divider-line"></span>
</div>
{% raw %}{% from "voyager/components/section-divider/section-divider.j2" import section_divider %}

{{ section_divider("Journal settings · 5", flush=True) }}
{{ section_divider("Review · 2") }}{% endraw %}

With severity dot

For inbox buckets — a coloured dot at the left indicates severity.

  <span class="section-divider-line"></span>
</div>
{% raw %}{{ section_divider("Critical · 2", severity="critical") }}
{{ section_divider("Warning · 2",  severity="warning") }}
{{ section_divider("Info · 1",     severity="info") }}
{{ section_divider("All clear · 1") }}{% endraw %}

Notes

  • The label is an <h2> — section dividers form genuine document structure.
  • Use section-divider-flush for the first divider in a list to drop the top margin.
  • Severity dot variants are decorative; the count or severity name in the label carries the meaning.
