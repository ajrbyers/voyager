Filter bar

Control row that sits above a list of results. Select dropdowns on the top, active filter chips below them showing the current selection — each chip has its own remove (×) and there's a "Clear filters" link when any filters are active.

Default

Filter:
Hassan × Review × Clear filters 
<div class="filter-bar" role="search" aria-label="Filter articles">
  <div class="filter-bar-controls">
    <span class="filter-bar-label">Filter:</span>
    <label class="visually-hidden" for="f-editor">Editor</label>
    <select class="filter-select" id="f-editor">
      <option value="">All editors</option>
      <option value="1" selected>Dr Amira Hassan</option>
    </select>
    …
  </div>
  <div class="filter-bar-chips">
    <span class="filter-chip">
      Hassan
      <a href="?…" class="filter-chip-remove" aria-label="Remove editor filter">×</a>
    </span>
    <a href="?" class="filter-bar-clear">Clear filters</a>
  </div>
</div>
{% raw %}{% from "voyager/components/filter-bar/filter-bar.j2"
   import filter_bar, filter_bar_controls, filter_select,
          filter_bar_chips, filter_chip, filter_clear %}

{% call filter_bar() %}
  {% call filter_bar_controls() %}
    {{ filter_select("f-editor", "Editor",
                     options=[{"value":"","label":"All editors"},
                              {"value":"1","label":"Dr Amira Hassan"}],
                     value=active.editor) }}
    {{ filter_select("f-section", "Section",
                     options=[{"value":"","label":"All sections"},
                              {"value":"article","label":"Article"}],
                     value=active.section) }}
  {% endcall %}

  {% call filter_bar_chips() %}
    {% if active.editor %}
      {{ filter_chip("Hassan", remove_url="?section=" ~ active.section) }}
    {% endif %}
    {% if active.section %}
      {{ filter_chip("Review", remove_url="?editor=" ~ active.editor) }}
    {% endif %}
    {% if active %}{{ filter_clear(url=folder_url) }}{% endif %}
  {% endcall %}
{% endcall %}{% endraw %}

Notes

  • The select dropdowns submit on change (HTMX or vanilla form submit) — the bar itself does no JavaScript.
  • Chips reflect active selections. Each chip's remove link should remove only that filter while preserving the others.
  • Hide the chips row entirely when no filters are active — the empty filter-bar-chips collapses to zero height automatically.
  • Pair with the folder list and article card for the inbox-style dashboard.
