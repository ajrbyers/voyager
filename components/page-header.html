Page header

The first thing the user reads on a page. Caption above, H1 in the middle, byline and status tags below. Sits inside the page content area, not the global header.

Default

Caption gives context (the section, type, ID). Title is the H1. Byline names the relevant person and adds dates. Tags surface status at a glance.

Article #1247 · Peer review

Reproducibility of LSTM forecasts for short-term electricity demand

Dr Adaeze Okafor · University of Lagos · submitted 4 March 2026

Placement

Place the page header inside the main content column. When using .layout-with-rail, the title sits next to the rail's first section so the right column starts at the same vertical position as the H1.

Accessibility

  • The title is the only H1 on the page.
  • Caption is a span, not a heading — it provides context, not hierarchy.
  • Tags are decorative status labels; the same information must be available in the byline or status section so colour isn't the only carrier.
