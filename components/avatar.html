Avatar

Circular initials block. Three sizes; six neutral colours assigned by hashing the user's identifier on the consumer side so the same person always gets the same colour. The avatar itself is decorative — always pair it with the user's name in the surrounding markup.

Sizes

AH MW SP 
<span class="avatar avatar-purple" aria-hidden="true">AH</span>
<span class="avatar avatar-md avatar-orange" aria-hidden="true">MW</span>
<span class="avatar avatar-lg avatar-green" aria-hidden="true">SP</span>
{% raw %}{% from "voyager/components/avatar/avatar.j2" import avatar %}

{{ avatar("AH", colour="purple") }}
{{ avatar("MW", size="md", colour="orange") }}
{{ avatar("SP", size="lg", colour="green") }}{% endraw %}

Colour pool

Six neutral colours: purple, orange, green, red, teal, pink. Assign by hashing the user's primary key so the same person always renders with the same colour.

AB CD EF GH IJ KL 
<span class="avatar avatar-md avatar-purple">AB</span>
<span class="avatar avatar-md avatar-orange">CD</span>
<span class="avatar avatar-md avatar-green">EF</span>
<span class="avatar avatar-md avatar-red">GH</span>
<span class="avatar avatar-md avatar-teal">IJ</span>
<span class="avatar avatar-md avatar-pink">KL</span>
{% raw %}{# Map a stable user property to one of the colours #}
{% set colours = ["purple", "orange", "green", "red", "teal", "pink"] %}
{% set colour = colours[user.pk % colours|length] %}

{{ avatar(user.initials, size="md", colour=colour) }}{% endraw %}
