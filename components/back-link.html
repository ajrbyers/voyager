Back link

Small GDS-inspired link with a leading caret. Sits at the top of detail pages above breadcrumbs (where they exist) or as the sole “back” affordance on simpler pages. Always points to the previous list view, not to the browser's back history.

Default

Back to peer review queue 
<a href="/inbox?folder=peer_review" class="back-link">
  Back to peer review queue
</a>
{% raw %}{% from "voyager/components/back-link/back-link.j2" import back_link %}

{{ back_link("Back to peer review queue", url="/inbox?folder=peer_review") }}{% endraw %}

Notes

  • Always specify a concrete destination — e.g. “Back to peer review queue” rather than “Back”.
  • Don't use javascript:history.back() — it's unreliable and confusing if the user arrived via deep link.
  • Pair with breadcrumbs only when both add value; on most editorial detail pages a back link alone is enough.
