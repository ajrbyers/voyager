Icon

Inline SVG icons hand-rolled in the Lucide style. Every icon shares a 24×24 viewBox, 2 px stroke width, round caps and joins, and uses currentColor so the icon picks up the text colour of the surrounding element.

Sizing

The base .icon class sizes to 1em — the icon scales with the surrounding font size. Use size modifiers to override.

icon-sm 14px icon-md 18px icon-lg 24px icon-xl 32px 
<svg class="icon icon-sm" …>…</svg>
<svg class="icon icon-md" …>…</svg>
<svg class="icon icon-lg" …>…</svg>
<svg class="icon icon-xl" …>…</svg>

Inside other components

The component CSS auto-sizes icons inside their containers — an icon inside a button is 14 px, inside an action-menu icon tile is 13 px, inside a sidebar nav link is 13 px, and so on. Just drop the SVG in and it takes the right size.

Decorative vs labelled

Icons that sit next to text are decorative — use aria-hidden="true" so screen readers don't double-announce. Icons used as the sole content of an interactive element need an accessible name — use role="img" with aria-label, or wrap the icon in a button with an aria-label and keep the icon hidden.

<!-- Decorative (text describes it) -->
<a href="/files">
  <svg class="icon" aria-hidden="true" …>…</svg> Manage files
</a>

<!-- Icon-only button (icon hidden, button labelled) -->
<button type="button" aria-label="Open menu">
  <svg class="icon" aria-hidden="true" …>…</svg>
</button>

<!-- Standalone informative icon -->
<svg class="icon" role="img" aria-label="Critical" …>…</svg>
{% raw %}{% from "voyager/components/icon/icon.j2" import icon %}

<a href="/files">{{ icon("file") }} Manage files</a>

<button type="button" aria-label="Open menu">{{ icon("menu") }}</button>

{{ icon("alert", title="Critical") }}{% endraw %}

Notes

  • Don't use unicode glyphs (☎ ⚙ ✓) — they render inconsistently across systems. Reach for an SVG icon.
  • Don't fill icons with a colour. They use fill: none and stroke currentColor; colour them via color on the parent.
  • Don't mix stroke widths within a page. 2 px is the system default; small variants use 2.2 or 2.4 for visual weight at 12–14 px.
  • To add an icon: drop its SVG paths into the _ICONS dict in components/icon/icon.j2 and the macro picks it up.
