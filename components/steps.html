Steps & action menu

Steps is a numbered process list with state — used to show where a piece of work sits in a multi-stage flow. Action menu is a vertical list of action links with leading icons, typically used in the right rail.

Steps

Add .is-done for completed steps and .is-current + aria-current="step" for the active step.

  1. Assign editor
  2. Plagiarism checkSimilarity 7%
  3. Invite reviewers3 invitations sent
  4. Collect reviews1 of 2 received
  5. Editorial decision
  6. Notify author
<ol class="steps">
  <li class="is-done">Assign editor</li>
  <li class="is-done">Plagiarism check<small>Similarity 7%</small></li>
  <li class="is-done">Invite reviewers<small>3 invitations sent</small></li>
  <li class="is-current" aria-current="step">
    Collect reviews<small>1 of 2 received</small>
  </li>
  <li>Editorial decision</li>
  <li>Notify author</li>
</ol>
{% raw %}{% from "voyager/components/steps/steps.j2" import steps, step %}

{% call steps() %}
  {{ step("Assign editor",     state="done") }}
  {{ step("Plagiarism check",  hint="Similarity 7%", state="done") }}
  {{ step("Invite reviewers",  hint="3 invitations sent", state="done") }}
  {{ step("Collect reviews",   hint="1 of 2 received",    state="current") }}
  {{ step("Editorial decision") }}
  {{ step("Notify author") }}
{% endcall %}{% endraw %}

Action menu

Vertical list of action links with leading icons. The icon glyph is decorative — use aria-hidden="true" so it isn't announced as a letter.

  • View metadata
  • Document management
  • New review round
  • Reviewers from CSV
<ul class="action-menu">
  <li><a href="#"><span class="action-menu-icon" aria-hidden="true">i</span> View metadata</a></li>
  <li><a href="#"><span class="action-menu-icon" aria-hidden="true">D</span> Document management</a></li>
  <li><a href="#"><span class="action-menu-icon" aria-hidden="true">+</span> New review round</a></li>
</ul>
{% raw %}{% from "voyager/components/action-menu/action-menu.j2"
   import action_menu, action_menu_item %}

{% call action_menu() %}
  {{ action_menu_item("i", "View metadata",        url="/metadata") }}
  {{ action_menu_item("D", "Document management",  url="/docs") }}
  {{ action_menu_item("+", "New review round",     url="/round/new") }}
  {{ action_menu_item("↑", "Reviewers from CSV",   url="/reviewers/csv") }}
{% endcall %}{% endraw %}
