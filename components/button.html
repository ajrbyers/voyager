Button

Every button uses GOV.UK Design System styling — flat, square, with a 2px black bottom shadow. The default button is the GOV.UK secondary (grey) and is used for almost every editorial action. Compose with one of the variants for primary commits or destructive actions.

When to use which

  • Default (grey) — table-row actions, secondary toolbar buttons, "View", "Send reminder". Use this for almost everything.
  • Primary (green) — the page's main commit action: "Make a decision", "Submit review". One per view.
  • Warning (red) — destructive primary: "Delete reviewer", "Withdraw submission". Use sparingly.
  • Start — entry point to a service or wizard. Rare; never inside an existing flow.

Variants

<button type="button" class="btn">Save draft</button>
<button type="button" class="btn btn-primary">Make a decision</button>
<button type="button" class="btn btn-warning">Delete reviewer</button>
{% raw %}{% from "voyager/components/button/button.j2" import button %}

{{ button("Save draft") }}
{{ button("Make a decision", kind="primary") }}
{{ button("Delete reviewer", kind="warning") }}{% endraw %}

Sizes

Size modifiers compose with any variant. Use btn-small for table-row actions and dense toolbars where the default size is too tall.

<button class="btn btn-small">Small</button>
<button class="btn">Default</button>
<button class="btn btn-large">Large</button>
{% raw %}{{ button("Small", size="small") }}
{{ button("Default") }}
{{ button("Large", size="large") }}{% endraw %}

Start CTA

Only used as the entry point to a service or wizard. Don't use it for in-page submits.

Start review 
<a href="/review/start" class="btn-start">Start review</a>
{% raw %}{{ button("Start review", kind="start", href="/review/start") }}{% endraw %}

Button group

Cluster of related buttons. Use .btn-group-vertical when stacking long-labelled options like Manager menu links.

<div class="btn-group">
  <button class="btn">Cancel</button>
  <button class="btn btn-primary">Submit decision</button>
</div>
{% raw %}<div class="btn-group">
  {{ button("Cancel") }}
  {{ button("Submit decision", kind="primary") }}
</div>{% endraw %}

Disabled state

Use disabled on <button>. Anchors styled as buttons cannot be disabled — remove or replace them instead.

Accessibility

  • Always set type="button" on real buttons that aren't form submitters.
  • Don't use a <div> with click handlers — use a <button>.
  • Focus styling is handled by the global yellow focus rule. Don't override it on individual buttons.
