Pagination

Previous / Next plus numbered pages, GDS-inspired. Sits at the foot of any list view — articles, users, news posts, email log. The current page is announced to assistive technology with aria-current="page".

Default

  • ‹ Previous Page 4
  • 1 … 4 5 6 … 12
  • Next › Page 6 
<nav class="pagination" aria-label="Pagination">
  <ul>
    <li class="pagination-prev">
      <a href="?page=4" rel="prev">
        <span class="pagination-direction">‹ Previous</span>
        <span class="pagination-label">Page 4</span>
      </a>
    </li>
    <li class="pagination-numbers">
      <a href="?page=1">1</a>
      <span class="pagination-ellipsis">…</span>
      <a href="?page=4">4</a>
      <span class="pagination-current" aria-current="page">5</span>
      <a href="?page=6">6</a>
      <span class="pagination-ellipsis">…</span>
      <a href="?page=12">12</a>
    </li>
    <li class="pagination-next">
      <a href="?page=6" rel="next">
        <span class="pagination-direction">Next ›</span>
        <span class="pagination-label">Page 6</span>
      </a>
    </li>
  </ul>
</nav>

{{ pagination(
    prev_url="?page=4", prev_label="Page 4",
    next_url="?page=6", next_label="Page 6",
    pages=[
      {"n": 1,  "url": "?page=1"},
      {"ellipsis": True},
      {"n": 4,  "url": "?page=4"},
      {"n": 5,  "current": True},
      {"n": 6,  "url": "?page=6"},
      {"ellipsis": True},
      {"n": 12, "url": "?page=12"},
    ]
) }}{% endraw %}

Notes

  • Use ellipses to collapse long ranges — show first, last, current, and one or two adjacent pages.
  • Always include rel="prev" and rel="next" on the prev/next links.
  • Wrap in <nav aria-label="Pagination"> so screen readers announce it as a landmark.
  • Omit Previous when on page 1; omit Next on the last page.
