Folder list

Sidebar navigation for inbox-style pages. Groups folders into labelled sections, each item carries an icon, name, and optional count. Counts can be coloured to flag critical or warning severity. Custom (user-created) folders show a colour dot and inline edit/delete affordances on hover.

Default

  • Workflow

  • Unassigned 7
  • Peer Review 14
  • Copyediting 5 
<nav class="folder-list" aria-label="Inbox folders">
  <div class="folder-list-search">
    <label class="visually-hidden" for="inbox-search">Search articles</label>
    <input type="search" id="inbox-search" class="form-input" placeholder="Search articles…">
  </div>
  <div class="folder-list-scroll">
    <ul>
      <li><h2 class="folder-list-section folder-list-section-first">Workflow</h2></li>
      <li><a href="?folder=unassigned" class="folder-item">
        <span class="folder-item-icon" aria-hidden="true">✉</span>
        <span class="folder-item-name">Unassigned</span>
        <span class="folder-item-count folder-item-count-warning">7</span>
      </a></li>
      <li><a href="?folder=peer_review" class="folder-item is-active" aria-current="page">
        <span class="folder-item-icon" aria-hidden="true">⌕</span>
        <span class="folder-item-name">Peer Review</span>
        <span class="folder-item-count folder-item-count-critical">14</span>
      </a></li>
    </ul>
  </div>
  <button type="button" class="folder-list-add">+ New folder</button>
</nav>

  {{ folder_item("Unassigned",  url="?folder=unassigned",

Custom folder with actions

User-created folders show a colour dot and inline edit/delete buttons that appear on hover.

  • Custom

  • Climate & energy issue 11 
<li class="folder-item folder-item-custom">
  <a href="?folder=custom:1" class="folder-item-link">
    <span class="folder-item-dot" data-colour="pink"></span>
    <span class="folder-item-name">Climate &amp; energy issue</span>
    <span class="folder-item-count">11</span>
  </a>
  <div class="folder-item-actions">
    <button class="folder-item-action-btn" aria-label="Edit folder">✎</button>
    <button class="folder-item-action-btn folder-item-action-btn-danger" aria-label="Delete folder">×</button>
  </div>
</li>
{% raw %}{{ folder_item(
    "Climate & energy issue",
    url="?folder=custom:1",
    count=11,
    custom=True,
    dot_color="pink",
    edit_url="/folders/1/edit",
    delete_url="/folders/1/delete"
) }}{% endraw %}

Notes

  • Counts use coloured backgrounds for critical/warning state, never colour alone — the count number is the primary signal; severity is supplementary.
  • The active folder gets aria-current="page" for assistive technology.
  • Custom folder actions are visually hidden until hover (or keyboard focus inside the row).
  • The search input uses the standard .form-input class — no special styling needed.
