Typography

Janeway's body type stack is GOV.UK's Helvetica/Arial. Chrome (top bar, sidebar links, tabs, breadcrumbs, buttons) uses the system font stack so dense lists look at home in the operating system. Code, IDs, file names and SHAs use the monospace stack.

Type stacks

  • --font-sans — body and content. "Helvetica Neue", "Arial", sans-serif
  • --font-system — chrome, dense data. -apple-system, BlinkMacSystemFont, "Segoe UI", Helvetica, Arial, sans-serif
  • --font-mono — code, IDs, files. ui-monospace, SFMono-Regular, "SF Mono", Menlo, Consolas, monospace

Scale

The scale is intentionally modest. We avoid GOV.UK's full display sizes — Janeway is dense; oversized headings consume real estate.

--fs-display 32pxEditorial decision
--fs-h1 24pxReproducibility of LSTM forecasts
--fs-h2 18pxReviewers
--fs-h3 16pxFiles in Round 2
--fs-lg 16pxBody large — for ledes
--fs-md 15pxBody medium — primary buttons
--fs-base 14pxBody — the default size for paragraphs and tables
--fs-sm 13pxBody small — sidebar links, breadcrumbs
--fs-xs 12pxHint / caption

Weights

  • 700 — used for h1–h6 and form labels (GOV.UK convention)
  • 600 — used for table-cell strong emphasis and active states
  • 500 — sidebar link active state, button labels
  • 400 — body default

Line height

  • --lh-tight 1.25 — headings
  • --lh-base 1.5 — body
