Layout

Janeway pages share a common shell — top header, left sidebar, main column — and a small set of inner layouts. Use the shell components for the chrome and the layout utilities to arrange page content.

App shell

The shell is two components: app-header at the top, then app-shell with the sidebar nav and main content side by side.

Layout with rail

The most common page layout: main content on the left (2/3) and a rail on the right (1/3, fixed 280px). Use the .layout-with-rail utility on the page wrapper.

Full-width layout

For dashboards and index pages where there is no rail. Use .layout-full on the wrapper.

Grid utilities

Equal-column grids for cards and tiles: .grid .grid-2, .grid-3, .grid-4. They collapse to a single column under 900px.

Spacing primitives

Two layout primitives in utilities.css:

  • .stack — vertical rhythm. Adds margin-top to every direct child after the first. Variants: .stack-sm, .stack-lg, .stack-xl.
  • .cluster — horizontal flex group with gap and wrap. Variants: .cluster-sm, .cluster-lg.

Breakpoints

Voyager uses two breakpoints, both max-width:

  • 900px — sidebar collapses to top, multi-column grids collapse to one.
  • 1100px — .layout-with-rail stacks the rail below main.
  • 600px — summary-list rows stack key/value/change vertically.
