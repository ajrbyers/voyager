Tokens

Voyager uses a two-tier token system. Raw tokens hold the underlying value; semantic aliases reference the raw tokens by role. Components reference the semantic tokens, never raw values.

Surfaces

--colour-canvas#ffffff
--colour-canvas-muted#f6f8fa
--colour-canvas-inset#eaeef2
--colour-surface-grey#f3f2f1

Borders

--colour-border-muted#d8dee4
--colour-border#d0d7de
--colour-border-strong#a4abb3
--colour-border-component#b1b4b6

Border tokens are tuned for WCAG 1.4.11 — borders that distinguish a UI component from its background must have ≥3:1 contrast against canvas.
